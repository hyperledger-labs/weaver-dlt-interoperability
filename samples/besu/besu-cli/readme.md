# besu-cli CLI

A CLI for besu-cli.

## Customizing your CLI

Check out the documentation at https://github.com/infinitered/gluegun/tree/master/docs.

## Publishing to NPM

To package your CLI up for NPM, do this:

```shell
$ npm login
$ npm whoami
$ npm lint
$ npm test
(if typescript, run `npm run build` here)
$ npm publish
```

# Commands
- asset init
- asset get-balance
- asset lock
- asset claim
- asset unlock
- asset is-locked
- asset-exchange
